## 🚀 About The Project

**RalphVirus** is a script designed to cause a little harmless chaos. Once executed, it intercepts keyboard inputs and randomly maps pressed keys to entirely different letters. Whether you are using it as an educational project to understand keyboard listeners or just to pull a prank on a friend, RalphVirus gets the job done silently and effectively.

> **⚠️ Disclaimer:** This project is intended for educational purposes and harmless pranks on your own machines or with explicit permission. Please use responsibly!

## 💻 System Requirements

Choose your preferred method of execution. RalphVirus is highly accessible and requires minimal setup.

### Option A: Python Script
Perfect for developers and tinkerers who want to read or modify the source code.
* **Engine:** Python 3.x or higher
* **Supported OS:** Windows, macOS, or Linux

### Option B: Standalone Executable (Recommended)
Perfect for plug-and-play pranking without needing a Python environment.
* **Supported OS:** Windows OS only
* **Dependencies:** None! (Works right out of the box)
